# Graph view

Every note here is a node; every link an edge. Depth is shown with
glyph size and brightness — near notes are bright `◉`, far ones fade
to `·`.

Orbit with `hjkl`, zoom with `J`/`K`, cycle nodes with `n`, open with
enter. `b` cycles knowledge bases (each top-level folder is one).
